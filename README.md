# typesript-exercises

## Цель

Различные опыты и эксперементы с typescript

## Описание

### async-wait

Пример использования async/wait в typescript. 

Запуск:

~~~shell
cd async-wait
npm install
npm run start
~~~

### kefir

Баловство с Кефиром. 

~~~shell
cd kefir
npm install
npm run start
~~~

### rxjs

Баловство с rxjs. 

~~~shell
cd rxjs
npm install
npm run go1
~~~
